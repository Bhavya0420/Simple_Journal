# Simple Journal App

A simple journaling web application that allows users to write, save, view, and manage personal journal entries. The project focuses on a clean UI, basic CRUD operations, and a clear separation between frontend and backend.

This project is suitable for beginners to understand full-stack development using a modern frontend setup and a Node.js backend.

## âœ¨ Features

* User authentication (Login & Signup)
* Create new journal entries
* View all saved entries
* Edit and update existing entries
* Delete journal entries
* Search journal entries by keywords
* Filter journal entries for easy access
* Light mode and dark mode support
* Clean and minimal user interface
* Environment-based configuration using `.env`
